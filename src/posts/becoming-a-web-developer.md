# Becoming a Web Developer

Learning web development has been one of the most challenging and rewarding things I have ever done.

When I first started learning HTML and CSS, I never imagined that I would eventually be building full-stack applications with React, Flask, PostgreSQL and APIs.

There is still a lot to learn, but every project teaches me something new.

## What I have learned so far

Some of the technologies I have been working with include:

- HTML
- CSS
- JavaScript
- React
- Python
- Flask
- PostgreSQL
- REST APIs

The most important thing I have learned is that becoming a developer is a process.

You do not need to know everything before you begin.
