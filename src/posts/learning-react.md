# What I Am Learning About React

React has completely changed the way I think about building websites.

Instead of creating one large page, React allows you to break an application into smaller reusable components.

For example:

- `Navbar`
- `Footer`
- `BlogCard`
- `BlogList`
- `BlogPost`

This makes it much easier to maintain and reuse parts of an application.

## Components

A simple React component might look like this:

```jsx
function Greeting() {
  return <h1>Hello, world!</h1>;
}
```
